# Title CTR Rules

Page titles are one of the highest-leverage SEO fields. They should match search intent and earn clicks.

## Core rule

For non-homepage SEO pages, do not append the brand unless the brand is part of the search intent.

Google often displays the site/domain separately. Brand suffixes can waste characters that could carry the keyword or benefit.

## Good title patterns

| Page type | Pattern |
|---|---|
| Product | `<Product> - <Category or Benefit>` |
| Use case | `<Use Case> - <Key Benefit>` |
| Free tool | `<Tool Name> - Free <Utility>` |
| FAQ | `<Question> - <Short Answer>` |
| Glossary | `<Term> - Definition, Examples & How to Use` |
| Comparison | `<Competitor> vs <Product> - <Year> Comparison` |
| Alternative | `Best <Competitor> Alternatives in <Year>` |

## Modifiers

Use when truthful:

- Free
- Online
- Fast
- Instant
- Examples
- Guide
- Tutorial
- Template
- Generator
- Calculator
- Preview

## Length

Aim for roughly 30-60 characters. Do not sacrifice clarity just to hit a number.

## Avoid

- generic titles like `Home`, `Welcome`, `Overview`
- unnecessary brand suffixes
- unsupported superlatives like `best` or `#1`
- clickbait that the page does not satisfy
- duplicate titles across indexable pages
