# SEO and GEO Rules

## Baseline SEO

Every indexable page should have:

- unique title
- unique meta description
- one H1
- canonical URL
- crawlable HTML content
- relevant internal links
- correct sitemap inclusion

## AI/GEO visibility

AI systems prefer pages that are:

- factual
- directly answer the question
- structured with clear headings
- include FAQs where useful
- cite stable product facts
- avoid hype and vague claims

Recommended sections:

- "What is this?"
- "Who is it for?"
- "How it works"
- "FAQ"
- commands/examples when applicable

## Schema

Use schema only when supported by visible content and product facts.

Common schema:

- `Organization`
- `WebSite`
- `WebPage`
- `SoftwareApplication`
- `WebApplication` for free tools
- `CollectionPage` for tool hubs/directories
- `BreadcrumbList`
- `FAQPage`

Do not add:

- fake reviews
- fake ratings
- fake offers/prices
- unsupported customer logos
