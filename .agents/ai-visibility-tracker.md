# AI visibility tracker

Last updated: 2026-05-09
Owner: Pagewell team
Cadence: Monthly manual check

Use this file to track whether Pagewell is mentioned or cited by AI answer systems for priority queries. Do not treat results as statistically stable; record the date, platform, query wording, citation URLs, and notable competitors.

## Monthly check workflow

1. Run each query in Google with AI Overviews when available, ChatGPT with search, Perplexity, Gemini, and Copilot.
2. Record whether Pagewell is cited, mentioned without citation, or absent.
3. Record which Pagewell URL is cited, if any.
4. Record competitors or adjacent tools cited.
5. Add notes about answer framing, missing facts, or pages that need stronger extractable sections.
